# Football Formations — Overview & Index

All formation definitions live in the category files below. This file covers shared concepts and quick-reference tables.

## Notation System

Formations are written as defenders-midfielders-forwards (e.g. 4-3-3 = 4 DEF, 3 MID, 3 FWD). When a formation has a distinct defensive or attacking mid layer, it gets its own number (4-2-3-1 = 4 DEF, 2 CDM, 3 AMF, 1 ST).

## ⭐ Default Formation: 4-3-3

Use **4-3-3** when no formation emerges as a clear winner from match data.

Positions: GK · RB · CB · CB · LB · CDM · CM · CM · RW · ST · LW
